# Tall Building Preliminary Structural Analysis Framework (SDOF + MDOF)

A computational structural framework for preliminary tall-building design, integrating global SDOF estimation with calibrated MDOF modal analysis.

## Features
- Preliminary structural sizing for tall buildings
- SDOF global period estimation
- MDOF modal analysis with first 5 mode shapes
- Zone-based plan visualization
- Report export
- Streamlit web interface

## Live App
[Open the web app](https://fbdjaoexq5qbz2zqflc8mb.streamlit.app/)

## Repository Structure
- `app.py` → main Streamlit application
- `requirements.txt` → dependencies
- `README.md` → project documentation

## Methodological Basis
The framework combines:

- **SDOF approximation** for rapid global dynamic estimation
- **MDOF shear-building model** for modal characterization
- **Zone-based structural visualization** for lower, middle, and upper building regions

## Outputs
- Target period
- Estimated period
- Structural stiffness
- Mode shapes
- Zone-based plan view
- Downloadable report
